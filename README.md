# TCC — Lucas Ivonovith - UMA ABORDAGEM ALGORÍTMICA PARA PROBLEMAS DE OTIMIZAÇÃO INTEIRA NÃO LINEAR COM RESTRIÇÃO ELIPSOIDAL

Este repositório contém os arquivos desenvolvidos durante a elaboração do Trabalho de Conclusão de Curso do discente Lucas Ivonovith da Universidade Federal do Recôncavo da Bahia (UFRB), incluindo a implementação computacional dos algoritmos propostos e os arquivos auxiliares utilizados na análise experimental.

## Ambiente de desenvolvimento

Os códigos foram implementados em linguagem Python em sua versão 3.12.7. O desenvolvimento foi realizado utilizando o ambiente Visual Studio Code

solver utilizado nos experimentos foi o Gurobi Optimizer em sua versão 12.0.1. Para a execução dos códigos, é necessária a instalação da biblioteca correspondente, bem como uma licença ativa do software.

## Sobre os experimentos

As planilhas presentes no repositório correspondem aos dados experimentais utilizados na elaboração das tabelas e análises apresentadas no Trabalho de Conclusão de Curso.

Os códigos referentes aos algoritmos AP e AL para instâncias de dimensão 5 foram implementados, porém não foram utilizados nos experimentos finais em razão das limitações e considerações metodológicas discutidas no trabalho.

## Observação

Os resultados e códigos aqui disponibilizados correspondem exclusivamente ao material desenvolvido para fins acadêmicos no contexto deste Trabalho de Conclusão de Curso.
